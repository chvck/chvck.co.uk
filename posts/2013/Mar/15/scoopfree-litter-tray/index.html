Scoopfree Litter Tray

Posted on 15 Mar 2013

This is a bit off the beaten path of bike related things that I usually write about. This post is about a cat litter tray system called Scoopfree, it's one of those things that actually makes a difference. As the name suggests there's no scooping cat poo with this machine, you simply insert a cartridge and off you go until it's time to fit a new cartridge.

The way that this thing works is that it has a sensor (light beam I think) that when crossed triggers what I'd called an event. Once this event has occured then the litterbox counts down for 20 minutes (light blinks to let you know) and then it self-cleans. The self-clean process is simply a rake that runs down the tray and deposits the cat poo, dirty litter etc... in a compartment and then it runs back up. You'd think that this would smell pretty bad but it doesn't! They send a special form of litter that absorbs the smell, obviously it does start to get a bit stinky if you leave replacing the cartridge too long.

A cartidge lasts about a month for us with just Tesla to worry about and each one costs about a tenner from Vets UK. At the end of a work day when you're tired it's nice to come home and not have to muck about with cat poo. It's expensive for what it is but I do like having it around, especially as Tesla still insists on attacking it every time that it rakes! We've had it a few months now too.
